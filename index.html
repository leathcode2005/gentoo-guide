Gentoo Installation Guide (OpenRC + UEFI + ext4)

This guide is intentionally minimal. It keeps only required steps for a standard OpenRC install on UEFI hardware.

1) Prepare Disk

Identify target disk:


lsblk -o NAME,SIZE,TYPE,FSTYPE,MOUNTPOINT

Start partitioning (replace `/dev/sda` as needed):


fdisk /dev/sda

Inside `fdisk`, create GPT + 3 partitions:


g
n
1
<Enter>
+1G
t
1
n
2
<Enter>
+8G
t
2
19
n
3
<Enter>
<Enter>
w

Resulting layout:

  • `/dev/sda1`: EFI System Partition
  • `/dev/sda2`: swap
  • `/dev/sda3`: root

2) Create Filesystems and Mount

Format partitions:


mkfs.vfat -F 32 /dev/sda1
mkswap /dev/sda2
swapon /dev/sda2
mkfs.ext4 /dev/sda3

Mount target root and EFI:


mkdir -p /mnt/gentoo
mount /dev/sda3 /mnt/gentoo
mkdir -p /mnt/gentoo/efi
mount /dev/sda1 /mnt/gentoo/efi

3) Download and Extract Stage3

Go to target root:


cd /mnt/gentoo

Sync time for HTTPS:


chronyd -q

Download OpenRC stage3 (use current filename from Gentoo mirror):


wget https://distfiles.gentoo.org/releases/amd64/autobuilds/current-stage3-amd64-openrc/stage3-amd64-openrc-<date>.tar.xz

Extract stage3 (run from within /mnt/gentoo):


tar xpvf stage3-*.tar.xz --xattrs-include='*.*' --numeric-owner

4) Configure Portage (make.conf)

Set compiler defaults before entering the chroot so build settings are in effect for the first Portage sync:


cat > /mnt/gentoo/etc/portage/make.conf <<'EOF'
COMMON_FLAGS="-march=native -O2 -pipe"
CFLAGS="${COMMON_FLAGS}"
CXXFLAGS="${COMMON_FLAGS}"
MAKEOPTS="-j$(nproc)"
EOF

5) Chroot

Copy DNS config:


cp --dereference /etc/resolv.conf /mnt/gentoo/etc/

Mount the necessary pseudo-filesystems:


mount --types proc /proc /mnt/gentoo/proc
mount --rbind /sys /mnt/gentoo/sys
mount --make-rslave /mnt/gentoo/sys
mount --rbind /dev /mnt/gentoo/dev
mount --make-rslave /mnt/gentoo/dev
mount --bind /run /mnt/gentoo/run
mount --make-slave /mnt/gentoo/run

Enter target system:


chroot /mnt/gentoo /bin/bash
source /etc/profile
export PS1="(chroot) ${PS1}"

Note: If using the official Gentoo install media, you may instead use arch-chroot /mnt/gentoo as a shortcut — it handles the pseudo-filesystem mounts automatically.

6) Portage and Profile

Install repository snapshot:


emerge-webrsync

Set profile:


eselect profile list
eselect profile set <openrc-profile-number>

7) Basic System Settings

Set timezone to Detroit (OpenRC method):


echo "America/Detroit" > /etc/timezone
emerge --config sys-libs/timezone-data

Set locale:


echo 'en_US.UTF-8 UTF-8' > /etc/locale.gen
locale-gen
eselect locale list
eselect locale set <en_US-utf8-number>
env-update && source /etc/profile

8) Kernel

Configure installkernel to use dracut:


mkdir -p /etc/portage/package.use
cat > /etc/portage/package.use/installkernel <<'EOF'
sys-kernel/installkernel dracut
EOF

Set root UUID for initramfs:


ROOT_UUID=$(blkid -s UUID -o value /dev/sda3)
mkdir -p /etc/dracut.conf.d
cat > /etc/dracut.conf.d/00-installkernel.conf <<EOF
kernel_cmdline=" root=UUID=${ROOT_UUID} rw "
EOF

Install firmware, kernel tooling, and kernel:


emerge --ask sys-kernel/linux-firmware sys-kernel/installkernel sys-kernel/gentoo-kernel

9) Configure System Files

Set hostname:


echo gentoo > /etc/hostname

Create fstab:


ESP_UUID=$(blkid -s PARTUUID -o value /dev/sda1)
SWAP_UUID=$(blkid -s PARTUUID -o value /dev/sda2)
ROOT_PARTUUID=$(blkid -s PARTUUID -o value /dev/sda3)
cat > /etc/fstab <<EOF
PARTUUID=${ESP_UUID}      /efi   vfat  umask=0077,tz=UTC  0 2
PARTUUID=${SWAP_UUID}     none   swap  sw                  0 0
PARTUUID=${ROOT_PARTUUID} /      ext4  defaults,noatime    0 1
EOF

Set hosts file:


cat > /etc/hosts <<'EOF'
127.0.0.1   gentoo.localdomain gentoo localhost
::1         gentoo.localdomain gentoo localhost
EOF

Set root password:


passwd

10) Networking and Required Services

Install and enable DHCP client:


emerge --ask net-misc/dhcpcd
rc-update add dhcpcd default

Install and enable logger + cron + time sync:


emerge --ask app-admin/sysklogd sys-process/cronie net-misc/chrony
rc-update add sysklogd default
rc-update add cronie default
rc-update add chronyd default

11) UEFI Boot (EFI Stub, No GRUB)

Install efibootmgr:


emerge --ask sys-boot/efibootmgr

Copy kernel and initramfs to ESP:


mkdir -p /efi/EFI/Gentoo
cp /boot/vmlinuz-* /efi/EFI/Gentoo/bzImage.efi
cp /boot/initramfs* /efi/EFI/Gentoo/initramfs.img

Create firmware boot entry:


efibootmgr --create --disk /dev/sda --part 1 --label "Gentoo" --loader "\\EFI\\Gentoo\\bzImage.efi" --unicode "initrd=\\EFI\\Gentoo\\initramfs.img root=UUID=${ROOT_UUID} rw"

12) Finalize and Reboot

Create daily user:


useradd -m -G users,wheel,audio,video,usb -s /bin/bash larry
passwd larry

Exit, unmount, reboot:


exit
cd /
umount -l /mnt/gentoo/dev{/shm,/pts,} 2>/dev/null || true
umount -R /mnt/gentoo
reboot
